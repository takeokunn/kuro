# チュートリアル

**学習指向**: 初心者がステップバイステップで学ぶためのガイドです。

Diataxis フレームワークにおけるチュートリアルは、読者を手取り足取り導きながら、一連の手順を通じてプロジェクトの基本を体験的に学ばせることを目的としています。チュートリアルでは「何ができるか」を教えるのではなく、「実際にやってみる」ことで理解を深めます。

## チュートリアル一覧

- **[Getting Started](./getting-started.md)** — kuro のインストールと初回起動
  - 前提条件の確認
  - ソースコードの取得とビルド
  - Emacs の設定
  - 最初のターミナルセッションの起動
  - 動作確認とトラブルシューティング

- **[Basic Usage and Key Bindings](./basic-usage.md)** — 基本的な使い方とキーバインド
  - ターミナルの基本操作
  - キーバインド一覧
  - 実践的な使用例 (Git、ファイル操作、プロセス管理)
  - kuro 固有の機能
  - トラブルシューティング

- **[First Customization](./first-customization.md)** — 最初のカスタマイズ
  - 基本的な設定方法
  - Face とテーマのカスタマイズ
  - 動作設定の調整
  - キーバインドのカスタマイズ
  - 高度なカスタマイズ例

## 学習の順序

初めて kuro を使用する方は、以下の順序でチュートリアルを進めることをお勧めします：

1. [Getting Started](./getting-started.md) — まずは環境構築から
2. [Basic Usage and Key Bindings](./basic-usage.md) — 基本的な使い方をマスター
3. [First Customization](./first-customization.md) — 自分好みにカスタマイズ

## トラブルシューティング

各チュートリアルにはトラブルシューティングセクションが含まれています。問題が発生した場合は、まず該当するチュートリアルのトラブルシューティングセクションを確認してください。
