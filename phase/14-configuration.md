# フェーズ 14: 設定システム

## 目的

ユーザーがすべての設定をカスタマイズできる

**期間:** 18週目

## 関連ドキュメント

- [docs/how-to/configure.md](../docs/how-to/configure.md)

## 実装方針

- すべてのユーザーオプションに `defcustom` を使用
- 設定変数:
  - `kuro-shell`: シェルプログラム (デフォルト: $SHELL または /bin/bash)
  - `kuro-scrollback-size`: スクロールバック行数 (デフォルト: 10000)
  - `kuro-font-family`: フォントファミリー
  - `kuro-font-size`: フォントサイズ
  - `kuro-color-*`: 色のカスタマイズ
- 各オプションの検証関数
- 実行時の再設定サポート
- 主要関数: `defcustom` 変数, `kuro--validate-config`

### タスク

- [ ] すべての defcustom 変数を定義
- [ ] 検証関数を実装
- [ ] 実行時の設定再読み込みをサポート
- [ ] カスタマイズグループを作成
- [ ] すべてのオプションをドキュメント化

## 受け入れ条件

### 手動テスト手順

1. `M-x customize-group kuro` → カスタマイズ画面を開く
2. `kuro-scrollback-size` を 5000 に変更 → 適用
3. 新しいスクロールバック制限が有効になっていることを確認
4. `kuro-shell` を `/bin/zsh` に変更 → kuro を再起動
5. zsh が実行されていることを確認
6. カスタム色を設定 → 色が更新されることを確認

**期待結果:** すべての設定オプションが正しく動作
